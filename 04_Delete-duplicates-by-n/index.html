Delete duplicates by n

For a given array, return a new array with a given number of duplicates

The returned array should match the order of the original array

 

Examples

      deleteNth([23, 7, 43, 43, 7, 41, 7, 26, 22, 7, 41, 7], 2) -> [23, 7, 43, 43, 7, 41, 26, 22, 41]
      deleteNth([1,2,5,7,8,6,4,5,5,1,1,2,3,3,3], 1) -> [1, 2, 5, 7, 8, 6, 4, 3]
