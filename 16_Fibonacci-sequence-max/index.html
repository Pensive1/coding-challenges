Fibonacci Sequence (Max value)

The Fibonacci sequence is the integer sequence where the first two terms are 0 and 1. After that, the next term is defined as the sum of the previous two terms.

Example 
        0, 1, 1, 2, 3, 5, 8, 13, 21, ...
Task

Generate a sequence up to n. The last value has to be less than or equal to n.

Output

        fib(5) => 0, 1, 1, 2, 3, 5
        fib(20) => 0, 1, 1, 2, 3, 5, 8, 13
